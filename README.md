# Employee Management System (EMS)

An in-progress backend system for managing employees, departments, and designations using ASP.NET Core Web API and Entity Framework Core.

---

## 📌 Project Status
🚧 In Progress (Development Stage)

---

## 🎯 Purpose
This project is built to practice and demonstrate backend development skills including:
- RESTful API design
- Database management using SQL Server
- CRUD operations
- Filtering and validation logic
- Clean architecture using ASP.NET Core Web API

---

## 🛠️ Tech Stack
- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQL Server
- C#

---

## 📁 Modules

### 1. Employee Module
- Create Employee
- Update Employee
- Delete Employee
- Get Employee list

### 2. Department Module
- Create Department
- Update Department
- Delete Department
- Get Departments

### 3. Designation Module
- Create Designation
- Update Designation
- Delete Designation
- Get Designations

---

## ⚙️ Features (Planned / In Progress)
- CRUD operations for all modules
- Input validation using Data Annotations
- Filtering APIs
- Pagination (future improvement)
- Unique constraints (e.g. department name, email, contact number)
